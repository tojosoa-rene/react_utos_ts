##### **React Testing Library (RTL)**



**What is RTL?**



* A popular tool for testing React applications
* Help ensure that components work as expected
* Help test how the app behaves from the user's perspective



**Important aspects:**



* Test behaviour rather than implementation
* Find elements the way users find them , i.e. by text, role, label
* Avoid testing internal state or component structure
* Test should mimic real user interactions
